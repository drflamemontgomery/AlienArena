# AlienArena

This Game was created by my brother and I 10 years ago using the HaxePunk Game Engine

# Compilation

This Game uses the lime utility for building

```bash
lime build linux
lime build windows
lime build mac
lime build html5 # The web build is slow and runs at 30fps
```

# Controls

## Player 1

movement: left, right
jump: up
shoot: down

## Player 2

movement: a, d
jump: w
shoot: q

## Player 3

movement: g, j
jump: y
shoot: h
